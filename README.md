# LinAlg Lib

A Linear Algebra library for C programs

## Usage

```
#include <stdio.h>
#include <stdlib.h>
#include "linalg.h"

int main () {
  ...
}
```

All tests are in the [main file](https://github.com/imsouza/linalg-lib/blob/main/tests/main.c).
